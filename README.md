🎓 EDUCARE – Online Learning Management System

📖 Overview
EDUCARE is a full-stack Online Learning Management System (LMS) built to simplify digital education for both students and educators.

The platform enables:
Students to enroll in courses, watch lectures, and track their progress.
Educators to create, manage, and deliver course content efficiently.
This project demonstrates a complete full-stack web application using modern technologies.

🚀 Key Features
👩‍🎓 Student
User registration and login
Browse available courses
Enroll in courses
Watch lecture content
Track learning progress

👨‍🏫 Educator
Create and publish courses
Upload lectures and learning materials
Manage enrolled students

⚙️ System
Secure authentication with Firebase
RESTful API backend
Responsive UI design
Course management functionality

🛠️ Technology Stack
Layer	Technologies
Frontend	React.js, HTML, CSS, JavaScript
Backend	Node.js, Express.js
Database	MongoDB
Authentication	Firebase Authentication
Deployment	Render
